Bootstrap è un framework di strumenti liberi per la creazione di siti e applicazioni web. Contiene modelli di progettazione basati su HTML, CSS e JavaScript

HTML (HyperText Markup Language) è un linguaggio di markup utilizzato per la creazione di pagine web e applicazioni online.

CSS (acronimo di Cascading Style Sheets) è un linguaggio di stile utilizzato per formattare il contenuto di una pagina web.
